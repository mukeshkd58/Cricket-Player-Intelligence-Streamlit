# Recruiter Pitch

This project demonstrates practical sports analytics and data science skills:

- real-data ingestion from Cricsheet JSON
- robust ball-by-ball parsing
- player summary feature engineering
- matchup analytics
- ML classification for dismissal/boundary/wicket probabilities
- professional Streamlit dashboarding
- cricket-specific tactical interpretation
- safe image sourcing from Wikimedia/Wikidata
- PDF report generation
- reproducible project structure

It is not a match prediction app. It focuses on workflows closer to cricket performance analysts, scouts, coaches, and academy decision-makers.
