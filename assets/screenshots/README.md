# Screenshot Checklist

This folder is reserved for real screenshots captured from the running Streamlit app.

Do not add fake screenshots, stock images, or mock UI renders.

Capture these files after running:

```bash
streamlit run app.py
```

Required screenshots:

- `home.png` — Home dashboard
- `batter_intelligence.png` — Batter Intelligence page
- `bowler_intelligence.png` — Bowler Intelligence page
- `matchup_analysis.png` — Matchup Analysis page
- `ml_predictions.png` — ML Predictions page
- `ai_tactical_report.png` — AI Tactical Report page
- `player_comparison.png` — Player Comparison page
- `export_report.png` — Export Report page

Recommended capture settings:

- Browser width: 1440px or wider
- Use realistic selections from the included real sample/full data
- Avoid cropping away the page title and key metrics
- Do not edit player statistics manually
